# Robock

ロボ子さんをデスクトップに住ませて無限に癒やされたかった


## Requirements

* Windows 10 10.0.17134.0
* .NET Framework 4.7.2 以降
* DirectX 11 ~
* Windows Display Driver Model に対応したビデオカード
* インターネット接続
* YouTube で 1080p 動画をフルスクリーンで見てもカクつかない程度のスペック


## Components

* Robock
* Robock.Background
* Robock.Shared
